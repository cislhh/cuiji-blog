# 项目截图说明

将项目截图放置在此目录下。

## 图片命名规范

建议使用项目ID作为前缀，例如：
- `ecommerce-1.jpg` - 电商平台第1张截图
- `ecommerce-2.jpg` - 电商平台第2张截图
- `admin-1.jpg` - 管理后台第1张截图

## 图片要求

- **格式**: JPG 或 PNG
- **封面图**: 建议尺寸 1600x900 (16:9)
- **截图**: 建议尺寸 1920x1080 或以上
- **文件大小**: 建议压缩到 500KB 以下
